# Gemini Image Generation Prompts - TLDR Music

## Design Guidelines
- **Color Palette**: Golden (#f59e0b, #fcd34d) and pure black (#000000)
- **Typography**: Red Hat Display font family
- **Style**: Apple-like minimalism - clean, spacious, elegant
- **Brand**: TLDR Music - India's Top 25 Charts

---

## OG Image (1200x630px)

### Prompt for Gemini:

Create a minimalist Open Graph social media image (1200x630px) for TLDR Music with an Apple-inspired aesthetic:

**Layout & Composition:**
- Pure black (#000000) background with subtle depth
- Generous white space, asymmetric balance
- Clean geometric elements with precise alignment

**Typography:**
- Primary text: "TLDR Music" in Red Hat Display Bold, 72pt, white color
- Secondary text: "India's Top 25 Charts" in Red Hat Display Medium, 36pt, golden (#fcd34d)
- Tertiary: "Weekly Updated Charts" in Red Hat Display Regular, 24pt, soft white (#E5E5E5)
- Left-aligned with 80px margin
- Letter-spacing: -0.02em (tight, modern)

**Visual Elements:**
- Single golden (#f59e0b) circular gradient element in top-right corner, subtle glow
- Minimal music waveform visualization at bottom, thin lines, 2px stroke, golden accent
- Simple music note icon (24x24px) in golden, positioned near brand name
- Thin golden hairline rule (1px) separating sections

**Brand Element:**
- Small badge/chip at bottom: "music.lumiolabs.in" in 16pt Red Hat Display Medium
- Badge: rounded rectangle, black background, golden border (1px), 8px padding

**Overall Feel:**
- Ultra-clean, premium, sophisticated
- 80% empty space for breathing room
- Precision alignment on 8px grid
- Soft shadows, no harsh elements
- Think Apple Keynote slide aesthetic

## Apple Touch Icon (180x180px)

### Prompt for Gemini:

Create a minimalist Apple touch icon (180x180px) for TLDR Music with iOS design language:

**Base:**
- Pure black (#000000) background
- No rounded corners (iOS handles this automatically)
- Centered composition

**Icon Design:**
- Single golden (#f59e0b to #fcd34d gradient) music note icon
- 80x80px size, perfectly centered
- Clean vector style, 3px stroke weight
- Subtle golden glow/halo effect (4px blur radius, 20% opacity)

**Typography:**
- "TLDR" text below icon
- Red Hat Display Black, 20pt, white color
- Letter-spacing: 0.05em (slightly loose)
- Positioned 8px below icon

**Details:**
- Minimal, single focal point
- High contrast for small sizes
- No additional decorative elements
- Think iOS app icon aesthetic - simple, recognizable, scalable

**Grid & Alignment:**
- All elements on 4px grid
- Optical center, not mathematical center
- 16px padding from edges

## Usage Instructions

1. Copy the desired prompt to Gemini
2. Specify exact dimensions: 1200x630 for OG, 180x180 for touch icon
3. Request PNG format with transparent edges if needed
4. May need to iterate 2-3 times for perfect alignment
5. Download and replace current images in project root

## Notes

- Emphasize "Apple-like minimalism" in your Gemini prompts
- Request "high contrast" for readability
- Specify "vector-clean edges" to avoid pixelation
- Ask for "generous white space" if first result feels cluttered
